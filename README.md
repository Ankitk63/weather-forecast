🌦️ Weather App

A real-time weather application built using HTML, CSS, JavaScript, and PHP.
The app fetches live weather data from a weather API and displays detailed weather information with smart weather suggestions in a modern responsive UI.

🚀 Features
 - Search weather by city name
 - Real-time temperature updates
 - Humidity, wind speed, and visibility details
 - Feels-like temperature display
 - Smart weather suggestions
 - Responsive modern UI
 - Live weather API integration

🛠️ Technologies Used
 - HTML5
 - CSS
 - JavaScript
 - PHP
 - Weather API
